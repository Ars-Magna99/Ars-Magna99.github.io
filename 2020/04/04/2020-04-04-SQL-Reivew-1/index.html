Concept 1: Relational Database

A relational database is a set of formally described tables from which data can be accessed or reassembled in many different ways without having to reorganize the database tables.

Concept 2: SQL

Structured Query Language.

A domain-specific language used in programming and designed for managing data held in a relational database management system, or for stream processing in a relational data stream management system.
