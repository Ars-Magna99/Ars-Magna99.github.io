Posted on 2020-03-26 | In Career

何时会需要A/B Testing?

对于一个初创公司，新的产品是快速打开市场、获得盈利的手段，此时对A/B Testing的需求并不会很高。企业真正需要A/B Testing的时候，是当其设计出一种能够快速迭代、且产品已经到了一种比较成熟的模式的时候。

到了这个时候，A/B Testing的作用才会越来越大。

所以，一般当初创公司的规模维持在10-15人时，A/B Testing并不会是一个重要的议题。

当公司规模逐渐扩展，到达50人左右时，A/B Testing 的概念会慢慢地适用。

当公司到达50-100人的规模时，A/B Testing 对于提升公司利润、影响力等指标的作用会越来越明显。

透过这个时间线，可以思考一下，自己在面试不同发展阶段的公司的时候，面试被问到A/B Testing的概率，以及重视程度都是不一样的。

A/B Testing 的目的？

统计上：假设某种statement是significant（显著）的，然后通过实验去证明其是否成立。

商业上：以旧的产品/网站UI设计为参照，我们想要测试某种新产品/网站的新UI设计是不是会对某些指标产生显著的影响。

A/B Testing 的目的并不是一次A/B Testing就能达到的，更多情况下会以迭代形式重复进行，不断改进产品线或者设计。

2020-03-26 Fixed some issues of the Hexo

Posted on 2020-03-26 | In Blog Development
  1. 自从更换Hexo主题至Next之后，就一直存在这样的一个问题：博客首页的按钮，如Archive, Home, About等，均能正常显示图标，但是点击后页面跳转都会出现问题而最后导致404。一开始也是试图自己去查找问题原因，但是试了很多次的结果也都不太对。

昨天晚上最后查了很久，发现有一些人也出现过同样的问题，他们也给出了相应的博文去解释。问题其实出在Next自带的配置文件中，有一些代码有不规范的地方，最终会导致这个404问题。

所以最终的解决方案是：

修改主题配置文件: /themes/_config.yml

将/和||之间的空格全部去掉即可。

  1. 第一个问题尚未解决之前，我也曾经尝试过给博文增加标签或者分类，但是发现增加了标签分类的博文在部署到线上之后会显示不出来。试了几次，把加上去的标签、分类删掉就会正常显示。 后来发现是tags和categories页面的配置是错误的，应该用layouts：XXX 而不是types: XXX。改了之后就正常显示。

其实这两个问题都属于使用Hexo过程中遇到的再简单不过、再正常不过的问题，但是却困扰了我很多天，但是最后都能解决。可见未来的时间里还是要好好学习技术方面的知识。

XGBoosting Class on DataCamp: Chapter 1

Supervised Learning

  • Relies on labeled data
  • You already have some understanding of the past behavior of what you are trying to predict

Supervised learning is always used to solve classification problems with binary results or multi-leveled results.
For example, we can predict whether a borrower will default his loan, this will be a binary classification problem because we either predict a borrower to “yes” or “no”. There could also be a scenario in which we are asked to recognize different kinds of birds on a picture. This will be a classification problem with multi-leveled results.

How to judge the performance of a model?

The most commonly used method is ROC curve and AUC.
The larger the area under the ROC model, the better the model is.

Sometims, Accuracy can be useful:
Accurady = TP+TN/(Total number of predictions made)

Some considerations of supervised learning

  • Features can be categorical or numerical
  • Numeric features should be scaled.
  • Categorical features must be encoded. The most commonly used method is “one-hot” encoding.

How to build a personal blog using Hexo and Github pages

Posted on 2020-03-21
  1. 下载node.js（从官网获取）
    Hexo必须要有node.js来支持。
  2. 进入Terminal

sudo su 进入root用户模式

  1. 利用npm安装hexo框架

    npm install -g hexo-cli

  2. 在想要放置博客文件的路径下生成文件夹

  3. sudo hexo init 初始化一个博客

  4. 生成新的文章：

hexo n "my first post"

会生成一个md格式的文件，可以选择用vim编辑，或者用typora打开以后编辑。

  1. 编辑完成文章以后，在终端输入

hexo clean

hexo g 以此让hexo为你的文章生成一篇html。
