XGBoosting Class on DataCamp: Chapter 1

Supervised Learning

  • Relies on labeled data
  • You already have some understanding of the past behavior of what you are trying to predict

Supervised learning is always used to solve classification problems with binary results or multi-leveled results.
For example, we can predict whether a borrower will default his loan, this will be a binary classification problem because we either predict a borrower to “yes” or “no”. There could also be a scenario in which we are asked to recognize different kinds of birds on a picture. This will be a classification problem with multi-leveled results.

How to judge the performance of a model?

The most commonly used method is ROC curve and AUC.
The larger the area under the ROC model, the better the model is.

Sometims, Accuracy can be useful:
Accurady = TP+TN/(Total number of predictions made)

Some considerations of supervised learning

  • Features can be categorical or numerical
  • Numeric features should be scaled.
  • Categorical features must be encoded. The most commonly used method is “one-hot” encoding.

How to build a personal blog using Hexo and Github pages

Posted on 2020-03-21
  1. 下载node.js（从官网获取）
    Hexo必须要有node.js来支持。
  2. 进入Terminal

sudo su 进入root用户模式

  1. 利用npm安装hexo框架

    npm install -g hexo-cli

  2. 在想要放置博客文件的路径下生成文件夹

  3. sudo hexo init 初始化一个博客

  4. 生成新的文章：

hexo n "my first post"

会生成一个md格式的文件，可以选择用vim编辑，或者用typora打开以后编辑。

  1. 编辑完成文章以后，在终端输入

hexo clean

hexo g 以此让hexo为你的文章生成一篇html。

参考文献：

  1. 程序羊YouTube教学视频（https://www.youtube.com/watch?v=erKYtw4Rfhk）
  2. www.codesheep.com
